# Readme

 <h2 align="center">Slide</h2>

<P  align="center">

Um projeto baseado no Kanban para a criação de tarefas e ordenar o estado atual entre fazer, fazendo e feito podendo movimentar livremente cada tarefa, editar e excluir.

A estrutura principal é um array de objetos onde cada objeto é uma coluna que contem suas informações que é modificada a todos os momentos pelo usuário.
</P>
